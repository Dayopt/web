# プランの使い方

プランはDayoptのタイムボクシングの中心となる機能です。会議、集中作業、個人的なタスクなど、「これからやること」をプランとしてカレンダーに配置することで、時間を意識的に使う習慣が身につきます。

## プランを作成する

プランの作成方法はいくつかあります。

| 方法 | 操作 |
|------|------|
| **カレンダーをクリック** | 空いている時間帯をクリックしてInspectorを開く |
| **カレンダーをドラッグ** | 時間範囲をドラッグして開始・終了時間を指定 |
| **右クリック** | 空いている時間帯を右クリックして「新しいプラン」を選択 |
| **「+」ボタン** | 次の空き1時間を自動で見つけて作成 |
| **キーボード** | `N` キーで現在時刻にプランを作成 |

作成したら、タイトルを入力し、時間を調整し、タグを追加して、`Cmd+Enter`（または `Ctrl+Enter`）で保存します。

## プランを編集する

カレンダーやサイドバーのプランをクリックすると **Inspector**（詳細パネル）が開きます。以下のフィールドを編集できます。

- **タイトル** — 入力と同時に自動保存
- **日付** — 別の日に移動
- **開始・終了時間** — 時間帯を調整（所要時間は自動表示）
- **タグ** — 分類用のタグを追加・削除
- **メモ** — リッチテキストで詳細を記述
- **繰り返し** — 定期的に繰り返す設定（後述）
- **リマインダー** — 開始前に通知

Inspector内では `Arrow Up` / `k` と `Arrow Down` / `j` で前後のプランに移動できます。

## ドラッグ&ドロップ

カレンダー上のプランは直感的にドラッグ操作できます。

- **移動** — プランをドラッグして別の時間帯に移す
- **リサイズ** — 下端をドラッグして終了時間を変更
- **スケジュール** — サイドバーの未スケジュールのプランをカレンダーにドロップ

ドラッグ中に `Escape` を押すとキャンセルできます。

## 繰り返しプラン

Inspector内の繰り返しアイコンをクリックして、自動的に繰り返すプランを設定できます。

| パターン | 例 |
|---------|-----|
| **毎日** | デイリースタンドアップ |
| **毎週** | 週次レビュー |
| **平日（月〜金）** | 朝のルーティン |
| **毎月** | 月次レポート |
| **毎年** | 年次振り返り |

**カスタム設定** では、さらに細かく指定できます。

- 頻度と間隔（例：2週間ごと）
- 特定の曜日（例：月・水・金）
- 終了条件 — なし / 指定日まで / N回後

繰り返しプランを編集・削除する際は、適用範囲を選択します。

- **この予定のみ** — この回だけに変更を適用
- **これ以降のすべて** — 今回と将来の予定に適用
- **すべての予定** — シリーズ全体に適用

## プランを完了する

タスクが終わったら、プランを完了にします。

- プランカードの **チェックボックス** を直接クリック
- または Inspector で **「完了にする」** をクリック

**「完了 + Record作成」** を選ぶと、プランを完了すると同時に、同じタイトル・時間・タグでRecordを作成できます。計画と実績を紐づけて振り返りに活用しましょう。

## リマインダー

プラン開始前に通知を受け取る設定ができます。

| 設定 | タイミング |
|------|-----------|
| 開始時 | プラン開始時に通知 |
| 10分前 | 直前のリマインド |
| 30分前 | 準備の時間 |
| 1時間前 | 余裕を持った通知 |
| 1日前 | 前日の確認 |
| 1週間前 | 事前の確認 |

## コピー&ペースト

右クリックメニューの「コピー」または `Cmd+C` でプランをコピーし、カレンダー上の任意の位置で `Cmd+V` でペーストできます。タイトル、所要時間、タグがそのまま引き継がれます。

## プランリストパネル

ヘッダーからサイドバーパネルを切り替えると、プランの一覧を表示できます。

- **検索** — タイトルで絞り込み
- **フィルター** — ステータス（未完了/完了）、スケジュール（予定あり/なし）、タグで絞り込み
- **並び替え** — 作成日・更新日・タイトル順
- **グループ化** — タグ別・ステータス別

リストからプランカードをカレンダーにドラッグして、スケジュールに入れることもできます。

## キーボードショートカット

| ショートカット | 動作 |
|--------------|------|
| `N` | 新しいプランを作成 |
| `Cmd/Ctrl + Enter` | プランを保存 |
| `Escape` | Inspectorを閉じる / ドラッグをキャンセル |
| `Arrow Up` or `k` | 前のプランに移動 |
| `Arrow Down` or `j` | 次のプランに移動 |
| `Cmd/Ctrl + V` | コピーしたプランをペースト |
