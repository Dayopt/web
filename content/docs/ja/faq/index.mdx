# よくある質問

## タイムボクシングとは？

タイムボクシングは、1日を固定の時間ブロックに区切り、各ブロックに特定のタスクを割り当てる時間管理手法です。ToDoリストから順番にこなすのではなく、「いつ」「どのくらい」取り組むかを計画します。Dayopt はこのプロセスをシンプルにし、振り返りで改善を促します。

## Dayopt は無料で使えますか？

はい。Free プランでは最大3プロジェクト、1GB のストレージ、基本的な分析、API アクセスが利用できます。クレジットカード不要で、ずっと無料で使えます。

## Pro プランには何が含まれますか？

月額 $5 で、無制限のプロジェクト、100GB ストレージ、高度な分析とカスタムレポート、チームコラボレーション、カスタム連携、優先サポートが利用できます。

## Google カレンダーと連携できますか？

はい。Dayopt は Google カレンダーと連携でき、タイムボックスが既存の予定と並んで表示されます。変更は双方向で同期されます。

## 精度スコアとは？

精度スコアは、計画したタイムボックスと実際の結果を比較します。例えば、2時間のディープワークを計画して1.5時間しかできなかった場合、そのギャップがスコアに反映されます。時間が経つと、見積もりの正確さやどこで計画が崩れやすいかのパターンが見えてきます。

## エネルギーマッピングとは？

タイムボックスを完了した後、オプションで集中度やエネルギーレベルを記録できます。時間が経つと、いつ最も生産性が高いか — ピーク時間帯、低下期、回復期 — がわかります。これにより、最も重要な作業を適切な時間に配置できるようになります。

## データのプライバシーは？

Dayopt はデータの販売や広告表示を行わず、AI モデルのトレーニングにもデータを使用しません。データは転送時（TLS 1.3）および保存時に暗号化されています。いつでもアカウントとすべてのデータを削除でき、30日以内に完全に消去されます。

## アカウントを削除できますか？

はい。設定 > アカウントから「アカウントを削除」をクリックしてください。すべてのデータは30日以内に完全に削除されます。

## データをエクスポートできますか？

はい。設定ページからプランとレコードを CSV または JSON でエクスポートできます。

## モバイルアプリはありますか？

Dayopt はデスクトップとモバイルの両方のブラウザに最適化された Web アプリケーションです。専用のモバイルアプリはロードマップに含まれています。

## Dayopt は誰が作っていますか？

Dayopt は個人開発者によって作られています。VC 資金なし、データ販売なし、広告駆動のビジネスモデルなし。あなたの時間とデータを尊重する、シンプルなプロダクトです。
