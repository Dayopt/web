# FAQ: 技術

## モバイルアプリはありますか？

現在はデスクトップとモバイルの両方のブラウザに最適化されたWebアプリケーションとして提供しています。PWA（Progressive Web App）としてホーム画面に追加でき、アプリに近い体験ができます。ネイティブアプリはロードマップに含まれています。

## どのブラウザに対応していますか？

Chrome、Firefox、Safari、Edgeなど主要なモダンブラウザすべてに対応しています。常に最新の2バージョンをサポートしています。

## 動作が重くなることはありますか？

パフォーマンスには特に気を配っています。Core Web Vitals 90+を維持するよう最適化しており、軽快な操作感を大切にしています。もし遅いと感じることがあれば、ぜひフィードバックをお寄せください。

## APIでどんなことができますか？

タイムボックスの作成・取得・更新など、Dayoptの主要な機能をプログラムから操作できます。自分だけの自動化ワークフローを構築したり、他のツールと連携させたりできます。

## Webhook は対応していますか？

現在のAPI機能と今後の拡張については、ドキュメントをご確認ください。ご要望があればフィードバックをお寄せいただければ、開発の優先順位に反映します。

## データのインポートは可能ですか？

他のツールからのデータ移行については、お気軽にお問い合わせください。可能な限りスムーズな移行をサポートします。

## 障害・メンテナンス情報はどこで確認できますか？

ステータスページやお知らせを通じて、サービスの状態を透明性をもってお伝えしています。計画メンテナンスは事前に通知します。

## アクセシビリティへの対応はどうですか？

WCAG 2.1 AA準拠を目標に開発しています。キーボードナビゲーション、スクリーンリーダー対応、十分なコントラスト比など、すべてのユーザーが使いやすいツールを目指しています。

## 多言語対応はしていますか？

現在、英語と日本語に対応しています。UIのすべてのテキストが両言語で利用可能です。今後の言語追加はユーザーの声を聞きながら検討します。

## 自社サーバーにホスティングできますか（セルフホスト）？

現在はクラウドサービスとしての提供のみです。セルフホスト版については、需要を見ながら検討していきます。
